Istalación de Sistemas Operativos "SO"

Se realiza una revisión de las caracteristicas de la maquina para dectar que sistema operativo soporta y cual se puede instalar.

Revisión Básica De Errores

Se realiza una revisión de las caracteristicas de la maquina, luego miramos el rendimiento que nos otorga en administracion de tareas, para lograr llegar a una conclucion. si esta muy lenta procedemos a realizar una limpieza general, cambiamos ram y disco duro.

Mantenimiento Del Equipo

Se realiza la respectiva revisión, se destapa y se desensambla el computador para hacerle una limpieza general, se le cambia pasta termica y se vuelve a ensamblar. Tambien se hace un backup de información que no se desea perder y finalizamos formatendo el computador, y restableciendo la informacion del backup.

Backups De Información

Se realiza una revisión de los archivos y decumentos guardados en el Disco duro para verificar el tamaño del Backup el costo va dependiendo de la cantidad de informacion en GB, y si es un disco me canico o solido debidoa que el proceso es mas lento en un disco mecanico.

Servicio De Revisión Remota

Se realiza una conexión Remota con las diferentes herramientas que nos lo permiten algunas de ellas son, TeamViewer, LogMeIn Pro, Remote Utilities, Zoho Assist, entre otros. Al tener la conexión con la maquina solicitamos permisos para revisar y acelerar el computador eliminando algunos archivos basura, activando los nucleos, ampliando la memoria virtual. Tambien se ralizan instalaciones de herramientas o sftware que el cliente desee o necesite.
